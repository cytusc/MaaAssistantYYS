# MaaYYS - 阴阳师自动化助手

一款基于 MAA 框架开发的阴阳师游戏自动化工具，帮助玩家轻松完成日常任务，解放双手，享受游戏乐趣。

## 🎯 项目简介

MaaYYS 是一款专为阴阳师玩家打造的自动化辅助工具。它可以模拟人工操作，自动完成游戏中的重复任务，让您有更多时间专注于游戏的策略和乐趣部分。

## ✨ 核心功能

### 日常任务自动化
- **大蛇副本**：自动挑战大蛇副本，支持智能选择层数，自动领取奖励
- **每日签到**：自动完成每日签到，领取签到奖励
- **体力消耗**：智能消耗体力，自动选择最优副本

### 活动任务支持
- 自动识别并参与游戏内各种限时活动
- 活动奖励自动领取
- 活动任务进度追踪

### 智能优化
- 支持多种分辨率适配
- 自动识别游戏界面状态
- 智能重试机制，确保任务完成

## 🛠️ 系统要求

- **操作系统**：Windows 10 或 Windows 11
- **游戏环境**：支持网易阴阳师官方客户端或主流模拟器
- **硬件要求**：普通家用电脑配置即可运行

## 🚀 使用方法

### 基本步骤

1. **下载安装**：获取最新版本的 MaaYYS 程序
2. **启动游戏**：确保阴阳师游戏已启动并处于主界面
3. **运行工具**：打开 MaaYYS，选择要执行的任务
4. **开始自动**：点击开始按钮，工具将自动完成所选任务

### 注意事项

- 请确保游戏窗口处于可见状态
- 建议在游戏设置中调整合适的分辨率
- 使用过程中请勿手动操作鼠标和键盘
- 合理安排自动化时间，遵守游戏规则

## 📋 安全说明

- 本工具仅用于辅助玩家完成重复性任务
- 使用前请确保已充分了解游戏的用户协议
- 建议适度使用，避免影响游戏体验和账号安全
